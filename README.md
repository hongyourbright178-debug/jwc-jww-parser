# AI図面自動チェックシステム（Task 2）

## 概要

JWW電気設備図面をAIが自動チェックし、エラー報告＋改善提案を出力するシステムです。

**入力**: JWWファイルが入ったフォルダ  
**出力**: 法令基準チェック結果 + シンボル検出・配置分析 + 具体的な修正提案

## 使い方

### 基本（図面チェック）
```bash
cd /mnt/c/jww/samples/code
python3 cad_checker_v7.py /mnt/c/jww/samples/after
```

### テスト実行（動作確認）
```bash
python3 test_checker.py /mnt/c/jww/samples/after
```
→ 13項目の自動テスト。全てPASSなら本番OK。

### 新しいプロジェクトで使う場合
```bash
python3 cad_checker_v7.py /path/to/new/project/jww/
```
JWWファイルを1つのフォルダにまとめて指定するだけ。

## チェック内容

### PART A: 基準チェック（法令・基準に基づく設備数量）
JWWファイル内のテキストから設備を検出し、法令基準と照合。

| チェック項目 | 根拠法令 |
|---|---|
| 火災感知器 | 消防法施行令第21条 |
| 誘導灯 | 消防法施行令第26条 |
| 非常照明 | 建築基準法施行令第126条の4 |
| 分電盤 | 電気設備技術基準 |
| 漏電遮断器(ELCB) | 電気設備技術基準第15条 |
| コンセント | 内線規程 |
| 照明設備 | 建築基準法第28条 |
| ナースコール | 医療法施行規則第16条（医療施設のみ） |
| 非常放送 | 消防法施行令第24条 |
| ... 他17項目 | 検出された設備に応じて自動追加 |

### PART B: 図面検図（シンボル検出・配置分析）
JWWバイナリから座標データを直接読み取り、以下を分析。

- シンボル検出: 円形・三角・四角・十字のパターンを認識
- ゾーン分析: 50mmグリッドで図面を分割、設備カバー率を計算
- 設備内訳: テキストからサブタイプを識別（ET/EET/防水など）
- 空白エリア検出: 設備が配置されていないゾーンを特定

### 出力例
```
❌ [CRITICAL] 2F 非常放送：シンボル数不足（検出=0, 期待≧3）
   💡 対応: 非常放送スピーカーの追加設置
   📖 根拠: 消防法施行令第24条
   📄 確認図面: 非常放送設備 平面図, 系統図

⚠️ [WARNING] 1F 照明設備：シンボル数不足（検出=6, 期待≧10）
   💡 対応: 照明器具の追加・配置見直し
   📖 根拠: JIS Z 9110照度基準
```

## 対応建物タイプ

病院、住宅、保育園、学校、事務所、工場、倉庫、ホテル、介護施設、その他。
建物タイプの指定は不要 — AIが図面内容から自動判定します。

## 動作環境

- Python 3.6以上
- 外部ライブラリ不要（標準ライブラリのみ）
- WSL / Linux / Mac で動作
